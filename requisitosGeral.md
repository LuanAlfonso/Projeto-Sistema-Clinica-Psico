# Lista de Requisitos – Geral

| ID | Tipo | Descrição |
| :--- | :--- | :--- |
| **RF01** | Funcional | O sistema deve permitir o agendamento/cancelamento de consultas online. |
| **RF02** | Funcional | O sistema deve enviar lembretes de confirmação automáticos via WhatsApp/E-mail. |
| **RF03** | Funcional | O sistema deve possuir módulo de prontuário eletrônico com acesso restrito. |
| **RF04** | Funcional | O sistema deve gerar relatórios de faturamento e controle de pagamentos. |
| **RNF01** | Não Funcional | O sistema deve ser responsivo (Web/Mobile). |
| **RNF02** | Não Funcional | Armazenamento de dados sensíveis deve seguir a criptografia padrão LGPD. |
| **RNF03** | Não Funcional | Tempo de resposta para carregamento da agenda não deve ultrapassar 2 segundos. |
